# Gear We Go

An easy-to-use calculator for chain, belt, and gear sizing.

## About

**Gear We Go** is a lightweight web-based tool that helps calculate basic parameters for:

* Chain drives
* Belt drives
* Gear sizing

Built using only HTML, CSS, and JavaScript.

## Features

* Simple and fast interface
* Chain drive calculations
* Belt drive calculations
* Gear ratio and sizing tools

## Technologies

* HTML
* CSS
* JavaScript

## How to Use

1. Open `index.html` in your browser
2. Enter required values
3. Get instant results
